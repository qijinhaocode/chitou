# ChiTou

Master interview knowledge, not just memorize it.

## What is ChiTou?

ChiTou is an AI-powered interview mastery system designed for software engineers.

Unlike traditional flashcard tools, ChiTou focuses on deep understanding through:

- Study
- Recall
- AI Evaluation
- Spaced Repetition

## Vision

Turn:

"I've seen this before."

into:

"I can explain this confidently in an interview."
